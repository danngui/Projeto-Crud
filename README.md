# Projeto-CRUD

Sistema desenvolvido para automatizar e facilitar o gerenciamento de usuários e produtos através de um painel administrativo moderno, oferecendo mais organização, controle e praticidade na administração das informações.

---

## 🚀 Funcionalidades

- Cadastro de usuários
- Edição e exclusão de usuários
- Gerenciamento de produtos
- Dashboard administrativo
- Sistema de navegação lateral interativa
- Pesquisa e organização de dados
- Interface moderna e responsiva
- Controle de status dos usuários
- Página de perfil do usuário
- Estrutura modularizada com HTML, CSS e JavaScript

---

## 🎯 Objetivo

Proporcionar uma aplicação CRUD completa para praticar e demonstrar conceitos de desenvolvimento web, permitindo o gerenciamento eficiente de usuários e produtos em uma interface intuitiva, organizada e funcional.

---

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- MySQL
